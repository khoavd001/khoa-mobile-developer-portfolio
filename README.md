# Vu Dinh Khoa Mobile Developer Portfolio

Static portfolio prototype for Vu Dinh Khoa, built from the supplied CV and live app store links.

## Contents

- `index.html` - single-page portfolio content and semantic structure
- `styles.css` - base layout, navigation, buttons, and hero visuals
- `sections.css` - portfolio sections, cards, timeline, skills, and footer
- `responsive.css` - tablet and mobile layout rules

## Local Preview

Open `index.html` directly in a browser, or run a small local server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Deployment

The site is designed for GitHub Pages with the root of the `main` branch as the publishing source.
